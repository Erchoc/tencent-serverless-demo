## Qrcode generator

[Demo](http://service-7kqwzu92-1251556596.gz.apigw.tencentcs.com/release/qrgen?q=yugasun.com)

### Usage

```shell
# 1. install dependencies
$ npm install

# 2. install scfcli global, refer to https://cloud.tencent.com/document/product/583/33449
$ pip install scf

# 3. deploy
$ scf deploy -f
```
